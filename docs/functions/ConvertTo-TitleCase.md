# ConvertTo-TitleCase

## SYNOPSIS

Convert a string to title case.

## SYNTAX

```powershell
ConvertTo-TitleCase [[-String] <String[]>] [<CommonParameters>]
```

## DESCRIPTION

Convert a string to title case.

## EXAMPLES

### EXAMPLE 1

```powershell
ConvertTo-TitleCase -String 'my string'

Returns the string \`My String\`.
```
